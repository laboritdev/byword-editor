# LabWord

**Vision:** A fast, minimalist Markdown editor for writers — Electron + TypeScript on macOS.

**Solves:** Reliable editing UX, Agent terminal input, and a maintainable typed codebase.

## Goals

- **Desktop (`desktop/`):** Primary LabWord app (Electron, CodeMirror, strict TypeScript)
- Predictable keyboard shortcuts, overlays, and file workflow
- Agent terminal and AI integrations

## Stack

- Electron, React, TypeScript (strict)
- CodeMirror 6 for editing
- Vitest + ESLint for quality

## Out of scope (for now)

- Swift/native macOS implementation (removed)
- Windows/Linux builds (macOS first)
